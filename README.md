🍕 Ultimate Pizza Ordering App

A blazing-fast, fully responsive pizza ordering Single Page Application (SPA) built with React, Vite, Tailwind CSS, and Redux Toolkit. This project demonstrates modern frontend best practices including modular architecture, optimized state management, and a polished user experience.

⸻

🚀 Features
	•	⚡ Lightning Fast: Powered by Vite for instant hot reloads and optimized builds.
	•	🎨 Tailwind CSS: Clean and responsive design with utility-first CSS.
	•	🔧 Redux Toolkit: Centralized and scalable state management.
	•	🧩 Modular Architecture: Component-driven development for easy maintenance.
	•	🛒 Cart Management: Add, update, and remove pizzas from the cart.
	•	📱 Responsive UI: Mobile-first layout compatible with all screen sizes.
	•	🔄 Real-Time Updates: Instant reflection of actions across components.
	•	🌐 Clean Routing: Built using React Router DOM for seamless navigation.
